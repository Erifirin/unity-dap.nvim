# unity-dap.nvim

Adaptation of [Unity for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=VisualStudioToolsForUnity.vstuc)
for nvim.

> [!WARNING]
> Work in progress.
> The plugin is in development stage.
